まとめ

今週は「先週の予告が現実になった一週間」。先週ベータだった TypeScript 7.0 が 6 月 18 日に RC へ進み、安定版が約 1 か月後（7 月中旬目安）に視野へ入った。同じ 18 日、先週「17 日予告」としていた Node.js のセキュリティリリースが実際に公開され、全サポート系（22.23.0 / 24.17.0 / 26.3.1）で 13 件の CVE、うち High が 2 件という具体像が判明。crypto と TLS の High はすべての稼働中ラインに影響するため、まずここを塞ぎたい。エコシステムでは Vite/Rolldown を擁する VoidZero の Cloudflare 傘下入りという統治面の地殻変動が続き、CSS は Interop 2026 / Baseline 2026 が新年度の焦点を固めた。

今週の重要ポイント

TypeScript 7.0、ベータから RC へ — 安定版が射程に

6 月 18 日、Microsoft が TypeScript 7.0 の Release Candidate を公開した。先週の本紙で取り上げたベータからの正式な前進で、RC は即日 npm から入手できる。Go へ移植したネイティブコンパイラは型チェックがおおむね 10 倍速とされ、安定版（GA）は RC から約 1 か月後、7 月中旬ごろが目安とされている。

あくまで「書き直し」ではなく「移植」で、型チェックの意味論は従来と同一を保つ点が引き続き強調されている。1 年超のプレリリース検証を経て、すでに数百万行規模のコードベースや CI で実用に供されており、日常のワークフローでの評価に十分耐えるという位置づけ。本番採用前の検証を始めるなら、まさに今が好機。

  • TypeScript 7.0 RC: The Go-Native Compiler Has Landed — Digital Applied

Node.js セキュリティリリース公開 — 全系統で 13 CVE、High が 2 件

先週「6 月 17 日に予定」と伝えた Node.js のセキュリティリリースは、6 月 18 日（木）に v22.23.0 / v24.17.0 / v26.3.1 として公開された。アドバイザリは WebCrypto・TLS・HTTP/2・HTTP・DNS・Permission Model などにまたがる 13 件の CVE をまとめており、大半は Medium / Low だが High が 2 件含まれる。

High の 1 件目は WebCrypto の整数オーバーフロー（CVE-2026-48933）。subtle.encrypt() に 2GiB の倍数長の入力が渡るとプロセスが異常終了し、DoS につながる。2 件目は TLS の認証バイパス（CVE-2026-48618）で、Unicode のドット区切りの正規化がリゾルバと検証側でずれることに起因し、ワイルドカード証明書の深さ検証をすり抜けられる。いずれも稼働中の全ライン（22 / 24 / 26）に影響するため、未パッチのまま回避できるバージョンは存在しない。即時のアップデートを推奨する。

  • Node.js June 2026 Security Releases: CVE Patch Guide — Digital Applied

Svelte 6 月号 — リアルタイム購読 query.live() が前進

SvelteKit のリモート関数まわりが実用度を上げた。新しい query.live(...) はサーバーからのリアルタイムデータを扱いやすくし、長寿命の購読を async-iterable として回せる。さらにリモートクエリをイベントハンドラ・非同期コールバック・モジュールスコープのどこでも await でき、キャッシュの重複排除がリアクティブ／非リアクティブ双方で共有されるようになった。

先週触れた TypeScript 6.0 対応・.run() 廃止に続く動きで、サーバー状態をフロントに引き込む際の「待てる場所」が一気に広がった点が今号の見どころ。テンプレート内で値を宣言できるようになるなど、書き味の改善も入っている。

フレームワーク・ランタイム

シグナル／コンパイラへの収束、2026 年の現在地

主要フレームワークは、細粒度リアクティビティ・サーバーファーストレンダリング・コンパイラ駆動の最適化・AI 支援ワークフローという 4 テーマへ収束しつつある。React は Compiler による自動メモ化で useMemo/useCallback の手書きを不要にし、Angular は Signals でバンドルを縮小、Vue は Vapor Mode で仮想 DOM を外す方向、Svelte は Runes で言語レベルに反応性を埋め込む。「どのフレームワークか」より「どのリアクティビティ実装を選ぶか」という議論軸が定着してきた。

採用比率では依然 React が約 47% で首位、Next.js・Angular・Vue が続き、Svelte は約 7%。Create React App は役目を終え、新規は Vite が標準ツールという整理も改めて確認された。

エコシステム・ツール

VoidZero が Cloudflare 傘下へ — Vite/Rolldown の統治が動く

6 月初旬、Evan You が 2023 年に立ち上げた VoidZero の Cloudflare による買収が報じられた。VoidZero は Vite（ビルドツール）・Rolldown（Rust 製バンドラ）・Oxc（パーサ／トランスフォーマ）・Vitest（テストランナー）という次世代ツールチェーンを束ねる中核で、フロントエンド基盤の統治体制に関わる動きとして今週も受け止め方の議論が続いている。

技術面では、3 月に出た Vite 8 が Rolldown を単一の Rust 製バンドラとして本採用済みで、プラグイン互換を保ちつつ大幅な高速化を実現している。OSS としての継続性とロードマップへの影響を、利用側としては当面ウォッチしておきたい。

セキュリティ

Node.js アップデートの当て方 — 優先順位の付け方

今回の Node.js リリースを受けて、フロントエンド／フルスタックチームが今すぐ確認したい点。①稼働中ランタイムと CI/CD で使う Node のバージョンを 22.23.0 / 24.17.0 / 26.3.1 以降へ引き上げる、②TLS 認証バイパス（CVE-2026-48618）はサーバー間通信の信頼境界に効くため、証明書検証に依存する経路を優先的に潰す、③WebCrypto の DoS（CVE-2026-48933）は大きな入力を扱う暗号処理がある場合に効くので、該当箇所の有無を棚卸しする。LTS・Current いずれも対象な点に注意。

デザイン・CSS

Interop 2026 / Baseline 2026 — 今年の相互運用の焦点が固まる

ブラウザ各社（Apple・Google・Igalia・Microsoft・Mozilla）が進める Interop 2026 は、20 のフォーカスエリアを掲げる（うち 15 が新規、5 が 2025 からの継続）。アニメーションタイムライン系プロパティ（animation-timeline / scroll-timeline / view-timeline）、HTML 属性値を CSS に取り込む attr()、contrast-color()、スクロール駆動アニメーション、View Transitions、<dialog> と :open 擬似クラス、popover="hint" などが並ぶ。

2026 年中に全コアブラウザで相互運用に達した機能は「Baseline 2026」として整理される。ライブラリで補ってきた領域がプラットフォーム標準へ移る流れが続いており、採用判断は webstatus.dev などで対応状況を確認しながら進めたい。
