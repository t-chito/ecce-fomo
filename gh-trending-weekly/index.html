まとめ

今週の週間トレンドは、先週まで続いた「エージェントに渡す中身（スキル・仕様）」の潮流がデザイン領域へ一気に流れ込んだ週になった。象徴は上位に並んだ二つの競合——「AI が生成した見た目にならない UI」を掲げる Nutlope/hallmark（週間 +6,573★）と、同じ狙いの pbakaus/impeccable（週間 +2,360★）で、いずれも 57／46 個の「slop 検出ルール」で AI の凡庸なデザインを機械的に潰す。Google の stitch-skills も加わり、デザインをスキルとして配る動きが一気に前景化した。もう一つの軸はドメイン業務の代行で、トレーディング（Vibe-Trading）・就職活動（ai-job-search）・Office 文書（OfficeCLI）と、具体的な仕事をまるごとエージェントに預けるツールが軒並み週間 +3,500〜5,400★ で伸びた。土台側では端末・ファイルを操る DesktopCommanderMCP、MCP とヘッドレスを備えて書き換え中の OpenCut、艦隊型 ADE の orca が続伸。先週の codex-plugin-cc・OmniRoute・herdr も再びランクインし、AI エージェント一色の傾向は今週も揺るがない。

AI に「デザインの目」を持たせる

Nutlope/hallmark — 「AI くささ」を 57 個の検査で潰すデザインスキル（週間 +6,573★ / 累計 1.2万★）

Claude Code・Cursor・Codex 向けのデザインスキルで、いかにも AI が作った風の見た目にならない Web UI を生成することを目的とする。「57 個の slop テストゲート」と自己批評を通し、LLM が陥りがちなデザインの初期値（凡庸なフォント選択やカード偏重のレイアウトなど）を避けるという。

組み込みの 20 テーマに加え、独自デザインが要るブリーフ向けのカスタムテーマを持つ。動作モードは「新規 UI を作る」「既存コードを監査する」「レイアウトを再設計する」「憧れのサイトからデザインの型を学ぶ」の 4 つで、各ページを自己完結した HTML + CSS として出力する。導入は npx skills add nutlope/hallmark か手動でのファイル配置。主体は CSS・HTML、ライセンスは MIT。エージェントに「仕様」を渡す先週までの流れが、いよいよ「美的判断」の領域にまで広がった象徴的なツールだ。

  • Nutlope/hallmark

pbakaus/impeccable — 23 コマンドと 46 検出ルールで UI 品質を底上げ（週間 +2,360★ / 累計 4.7万★）

hallmark と狙いを同じくする、AI コーディングエージェント向けのデザイン指針システム。構造化したデザイン文脈・コマンド・品質チェックを与え、AI が生成するフロントエンドの質を引き上げる。単一の /impeccable コマンドに audit・polish・critique・animate など 23 のサブコマンドをぶら下げる構成を取る。

初回の /impeccable init でプロダクトとデザインの文脈を PRODUCT.md／DESIGN.md に取り込み、以後はそれを参照する。API 呼び出しなしでデザインのアンチパターン（多用されるフォント、コントラスト不足、カード偏重など）を見つける 46 個の決定論的な検出ルール、ブラウザで結果を見ながら詰めるライブ反復モード、対応 IDE に問題を表示するデザインフックを備える。Cursor・Claude Code・Copilot・Gemini CLI・Codex CLI ほか幅広いツールに対応。主体は JavaScript、ライセンスは Apache 2.0。hallmark と並び、「AI のデザイン初期値をどう矯正するか」が今週の中心テーマになったことを示す。

  • pbakaus/impeccable

google-labs-code/stitch-skills — Google Stitch と組むデザイン系スキル集（週間 +1,090★ / 累計 7.6千★）

Google Labs による、Agent Skills のオープン標準に沿ったスキルとプラグインの集合。Google Stitch と連携し、Claude Code・Cursor など複数のコーディングエージェントを横断して動く。コードとビジュアルデザインの相互変換や、Stitch デザインからの本番向けコンポーネント生成を担う。

プラグインは 3 系統で、コードからデザインを起こす・画面を生成する・デザインシステムを管理する「Design」、React／React Native コンポーネントや Remotion 動画・ダッシュボードを作る「Build」、DESIGN.md 生成やプロンプト強化・デザインの規約検証を行う「Utilities」に分かれる。利用には Stitch MCP サーバーをエージェント環境に構成しておく必要がある。主体は TypeScript、ライセンスは Apache 2.0。大手が公式にデザイン系スキルを配り始めた点で、hallmark・impeccable のコミュニティ発の動きと呼応する。
