まとめ

今週は「AI エージェント向けスキル」カテゴリが週間トレンドを席巻した。Claude Code や MCP エコシステムの普及を背景に、エージェントに新しい能力を付与するプラグイン型リポジトリが次々と数万スターを獲得している。同時に、LLM への入力トークンを 60〜95% 削減するという chopratejas/headroom が 11,000 超のスターを集め、AI コスト削減への強い関心を示した。ビッグテック側では Apple が Mac ネイティブの Linux コンテナツールを公開し大きな注目を集めたほか、Microsoft の markitdown も引き続き高い人気を維持している。

AI エージェントスキル & 自律型エージェント

mvanhorn/last30days-skill — マルチソース調査をワンコマンドで（週間 +12,422★）

Reddit・X・YouTube・Hacker News・Polymarket・Web を横断して任意トピックを調査し、自己完結型の HTML ブリーフを生成する AI エージェントスキル。従来の検索エンジンではなく、アップボートや「いいね」など実際のユーザーエンゲージメントでコンテンツをスコアリングする点が特徴だ。ELI5 モード（平易な言葉で要約）や Best Takes（バイラルな引用を抽出）なども搭載しており、調査のオーバーヘッドを大幅に削減できる。累計 40,000 スター超という数字が示すように、エージェントスキル市場の中心的な存在になりつつある。

  • mvanhorn/last30days-skill

NousResearch/hermes-agent — 経験から自己成長する AI エージェント（週間 +10,733★）

Nous Research が公開した自己改善型 AI エージェント。会話を重ねるごとにスキルを自動生成・永続化し、ユーザープロフィールを段階的に構築していく「学習ループ」が核心機能だ。Telegram・Discord・Slack・WhatsApp・Signal・CLI など多様なインターフェイスから利用でき、200 以上の LLM プロバイダーに対応。ローカル・Docker・クラウド（Modal、Daytona）いずれの環境でも動作する。MCP 統合やスキルシステムも備え、エージェント開発基盤としての活用が広がっている。

  • NousResearch/hermes-agent

Leonxlnx/taste-skill — AI 出力の「味」を改善するスキル（週間 +8,413★）

AI が生成するコンテンツが退屈で画一的になりがちな問題を解決するためのスキル。具体的な指示よりも「文体の感覚」を注入することで、出力に個性と読み応えを持たせることを目指している。Claude Code ユーザーが手軽に導入できる設計で、特にライティング用途での評判が高い。シンプルな実装ながら週間 8,000 超のスターを集めた背景には、AI 出力のクオリティへの幅広い不満があると見られる。

  • Leonxlnx/taste-skill

Panniantong/Agent-Reach — API 不要でインターネット全体にアクセス（週間 +5,186★）

Twitter/X・Reddit・YouTube・GitHub・LinkedIn・RSS など 10 以上のプラットフォームへのアクセス能力を AI エージェントに追加するツール。有料 API を使わず、オープンソースのバックエンドと無料エンドポイントを組み合わせる設計が特徴だ。プラットフォームごとにプライマリ／フォールバックを自動切り替えし、agent-reach doctor コマンドで接続状況を診断できる。Claude Code・Cursor・Windsurf など主要エージェント環境と互換性がある。

  • Panniantong/Agent-Reach

LLM 開発ツール & フレームワーク

chopratejas/headroom — トークンを 60〜95% 削減するコンテキスト圧縮ツール（週間 +11,282★）

LLM に到達する前にツール出力・ログ・RAG チャンク・会話履歴などあらゆるコンテキストを圧縮し、同等の回答品質を維持しながらトークン消費量を大幅に削減する。Python／TypeScript ライブラリ、ローカルプロキシサーバー、MCP サーバーの 3 形態で導入でき、Claude Code・Cursor・Aider・GitHub Copilot CLI などに対応。CCR（可逆圧縮）により元データをキャッシュして必要時に復元する仕組みも持つ。AI エージェントの利用コストが課題になるにつれ、インフラレイヤーでの効率化ツールへの需要が高まっている。

  • chopratejas/headroom

microsoft/markitdown — あらゆるファイルを Markdown に変換（週間 +7,280★）

Word・Excel・PowerPoint・PDF・HTML・画像など多様な形式のファイルを Markdown テキストに変換する Microsoft 製 Python ライブラリ。LLM のコンテキストに文書をそのまま流し込む用途で広く使われており、AI パイプラインの前処理として定番ポジションを確立しつつある。今週も 7,000 超のスターを獲得し、ロングテールの継続的な支持を集めている。

  • microsoft/markitdown

CopilotKit/CopilotKit — エージェント対応 React フロントエンドスタック（週間 +2,751★）

React・Angular・Slack などに対応した AI エージェント向けフロントエンドライブラリ。チャット UI・エージェントへの状態共有・エージェント制御コンポーネントなど、いわゆる「Agentic UI」構築に必要なものをまとめて提供する。バックエンドの LLM 選択に依存しない設計で、既存の React アプリにエージェント機能を後付けするユースケースに向いている。

  • CopilotKit/CopilotKit

aaif-goose/goose — Rust 製の拡張可能 AI エージェント（週間 +2,509★）

ローカル環境でソフトウェアのインストール・実行・テストまで行えるオープンソース AI エージェント。Rust 製で軽量・高速な動作が特徴で、MCP サーバーとのネイティブ統合もサポート。コーディング補助だけでなく、コマンド実行やシステム操作まで行える「フルエージェント」としての利用が想定されている。

  • aaif-goose/goose
