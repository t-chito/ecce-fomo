2026年7月15日（水） 過去1週間

まとめ

先週「7 月 9 日に解禁」と予告された GPT-5.6（Sol / Terra / Luna）が予定どおり一般公開され、今週の主役はローンチそのものよりその後の需要爆発と価格戦争に移った。Sam Altman は「Sol の伸びは異常」「エージェント製品の利用が 1 週間で 2.5 倍」と連投し、推論チームが増設に追われる状況を明かしつつ、「Sol は Fable の半額・トークン効率 2 倍、四半分の価格で届けられて嬉しい」と Anthropic を名指しで挑発した。Anthropic は Fable 5 の利用枠と Claude Code のレート 50% 増を 7 月 19 日まで再延長して応戦し、フラッグシップ露出の消耗戦が続く。同時に、Sol Ultra が 64 サブエージェントで50 年来の数学予想を 1 時間未満で「証明」（未査読）、OpenAI 自身が SWE-Bench Pro を「3 割が壊れている」と監査して推奨撤回、さらに Demis Hassabis がフロンティア AI の標準化機関を提言と、モデルの中身・評価・統治の三方向で大きな動きが重なった一週間だった。

今週の重要ポイント

GPT-5.6 が一般公開 → 需要が爆発、ChatGPT Work も同時投入

OpenAI は予告どおり 7 月 9 日に GPT-5.6 Sol / Terra / Luna を ChatGPT・Codex・API へ一斉ロールアウトし、あわせて Codex と GPT-5.6 を土台にした汎用エージェント ChatGPT Work（数時間の作業を最後までやり切る）と、新世代の音声モデル GPT-Live を投入した。公開直後から利用が急伸し、Sam Altman は「Sol の伸びは異常」「エージェント製品（Codex・ChatGPT Work）の利用がこの 1 週間で 2.5 倍」「推論チームが英雄的に増設しているが、近いうちに供給がきしむ可能性がある」と相次いで投稿。単なる公開ではなく、実利用が一段跳ねた週になった。

  • OpenAI — GPT-5.6: Frontier intelligence that scales with your ambition
  • OpenAI (X) — Introducing ChatGPT Work, an agent powered by Codex and GPT-5.6
  • Sam Altman (X) — 5.6 sol growth is insane; possible hiccups soon
  • Sam Altman (X) — 2.5x increase in usage of our agentic products in the last week

価格・効率の全面戦争 — 「Fable の半額・四半分の価格」対「7 月 19 日まで延長」

今週の業界最大のテーマは価格と効率の正面衝突だ。Altman は「GPT-5.6 Sol は多くのタスクで Fable の半額・トークン効率およそ 2 倍、同じ仕事を四半分の価格で届けられて嬉しい」と Anthropic を名指しで挑発。ラインナップ全体でも Terra は GPT-5.5 級を約半額、Luna は最高推論設定の GPT-5.5 を上回りつつ 25 分の 1 のコストと、値下げ圧を全段で効かせた。これに対し Anthropic は、7 月 12 日で切れる予定だった Fable 5 の全有料プラン開放と Claude Code の週次レート 50% 増を 7 月 19 日まで再延長して応戦。二大ラボがフラッグシップの露出と単価の両方で削り合う構図が、先週から一段激化した。

  • Sam Altman (X) — Sol is half the price and ~2x token efficient vs Fable; happy to deliver at one-quarter the price
  • Claude (X) — Extending Fable 5 access and 50% higher Claude Code limits through July 19
  • OpenAI (X) — GPT-5.6 Luna outperforms GPT-5.5 at 25x less cost

Sol Ultra、50 年来の数学予想を 64 サブエージェントで「証明」（未査読）

7 月 10 日、GPT-5.6 Sol Ultra が 1973 年提起の Cycle Double Cover 予想（橋のないグラフは各辺をちょうど二度覆うサイクル集合を持つ、という 50 年来の未解決問題）の証明を、最大 64 の並列サブエージェントを「積極的・動的に」束ねて 1 時間未満で生成したと報告された。序盤はあえて多様な定式化・代数的角度・構造帰納を各エージェントに探索させる設計で、マルチエージェントの計算物量を数学に振り向けた点が注目された。ただし証明は未査読で、この予想は過去に何度も誤った証明を集めてきた経緯があり、コミュニティの検証待ちだ。ゲームやコード生成を超える知的成果になり得るかが焦点になっている。

  • Greg Brockman (X) — 50-year-old math conjecture solved with Sol Ultra
  • The Decoder — GPT-5.6 Sol Ultra reportedly solves a 50-year-old math problem in under an hour

OpenAI、SWE-Bench Pro を「3 割が壊れている」と監査し推奨撤回

OpenAI は 7 月 8 日、広く使われるコーディング評価 SWE-Bench Pro を監査し、公開 731 タスクのうち人手レビューで 34.1%（249 件）、自動パイプラインで 27.4%（200 件）が「壊れている」と判定、フロンティアの実力を正しく測れないとして従来の推奨を正式撤回した。5 名のソフトウェアエンジニアと調査エージェントによる検証で、わずか約 5 か月前（2026 年 2 月）に自ら旧 SWE-bench Verified から移行を促した推奨を巻き戻した形。先週は Anthropic がマルチモデル構成の効果を SWE-bench Pro の数字で示していただけに、業界共通の物差しが揺らいだことの含意が大きい。

  • OpenAI (X) — We audited SWE-Bench Pro and found 30% of tasks broken
  • The Stack — OpenAI wants new benchmarks to replace "broken" SWE-Bench Pro

Hassabis、フロンティア AI の標準化機関（FINRA 型）を提言

Google DeepMind の Demis Hassabis が、個人マニフェスト「A Framework for Frontier AI and the Dawning of a New Age」で、フロンティアモデルを審査する米主導の標準化機関の創設を提唱した。ウォール街を監督する FINRA を範に、当初は各ラボが公開の最大 30 日前に自主的にモデルを共有し、サイバー・生物・「欺瞞」能力の安全性を検査。運用が「有効かつ堅牢」と示せれば、米市場での配備に合格を義務づける方向へ formalize し得るとする。対象は「原産国やオープン／クローズドを問わず」全フロンティア級。Sam Altman も「思慮深い提案」と反応し、モデル競争と並行して統治の議論が前面に出た週になった。

  • Axios — Demis Hassabis calls for a U.S.-led global AI watchdog
  • TechCrunch — DeepMind CEO calls for an independent standards body to regulate frontier AI
  • Sam Altman (X) — this is a thoughtful proposal from demis

モデルリリース・普及

GPT-5.6 の三段構成が本番配備へ — 価格は Sol / Terra / Luna で明快に

GPT-5.6 は Sol（100 万トークンあたり入力 $5 / 出力 $30）、Terra（$2.50 / $15、GPT-5.5 級を約半額で）、Luna（$1 / $6、最速・最安）の三段で公開された。Sol は BrowseComp 92.2%、OSWorld 2.0 で 62.6% と新記録を主張し、OSWorld では Opus 4.8 を上回りながら出力トークンを 85% 削減したとする。Microsoft 365 Copilot でも GPT-5.6 が既定の推奨モデルになり、ヘルス領域では「医師が GPT-5.6 の回答に見つけた不備が医師自身の回答より少なかった」との評価も示された。フロンティアの一般公開が、価格表・クラウド既定・業種別評価まで含む「面」の更新として動いた。

  • OpenAI (X) — Sol, Terra, and Luna are rolling out in ChatGPT, Codex, and the API
  • Sam Altman (X) — GPT-5.6 is now the preferred model in Microsoft 365 Copilot
  • Sam Altman (X) — physicians found fewer flaws in GPT-5.6 responses than physician-written ones

追い上げも活発 — Grok 4.5 が無料枠へ、Cognition は SWE-1.7 を公開

フロンティアの追走組も動いた。xAI は Grok 4.5 を無料枠で試せるようにし（Grok Build を X / Grok アカウントで利用可）、Perplexity の Computer 評価で「フロンティア各モデル中で最高スコアを競争力ある価格で」達成したと主張。API は入力 $2 / 出力 $6 と、GPT-5.6 Terra と同水準の低価格帯に置いた。Cognition は自社最強という SWE-1.7 を発表——オープンな Kimi K2.7 を土台に 1000 tok/s で動き、フロンティア上位と数点差、コストは一部という位置づけ。中国系ベースモデルの人権関連の拒否・偏りを問題視し、信頼性・中立性を重点に再学習したと説明した。低価格×高速の層が一段厚くなっている。

  • Grok (X) — Grok 4.5 is now available to try on the free tier
  • Cognition (X) — Introducing SWE-1.7, our most capable model yet, at 1000 tok/s
  • Cognition (X) — SWE-1.7 trained specifically for trustworthiness on top of Kimi K2.7

推論の経済性

「高価なモデルほど安く上がる」— Devin Fusion が Fable 5 を採用

先週から続く「賢さをどう経済的に配線するか」の議論に、実測が積み上がった。Cognition は Devin Fusion に Fable 5 を組み込み、「単価はより高いモデルなのに、委譲と推論チェーンの効率化でタスクあたりコストは Opus 4.8 より低かった」と報告。同社は GPT-5.6 も Devin に載せ、FrontierCode 1.1 Extended では「Sol が次点モデルの約半分のコストで最高性能に達した」とする。単体のトークン単価ではなく、オーケストレーションを含めた「タスク単位の総コスト」でモデルを選ぶ見方が定着してきた。OpenAI の価格攻勢は、この総コスト競争を一段前に進めている。

  • Cognition (X) — Devin Fusion now incorporates Fable 5; lower cost per task than Opus 4.8
  • Cognition (X) — GPT-5.6 in Devin: Sol reaches top performance at ~half the cost of the next best model

エージェント・ツール

開発環境が「エージェント前提」へ — ブラウザ内蔵・多人数編集・ACE 構想

エージェントの作業面（コーディング環境）が各社で更新された。Claude Code はデスクトップにアプリ内ブラウザを搭載し、ドキュメントやデザインをサンドボックス下で読み・クリック・操作できるように。Artifacts は公開共有とマルチプレイヤー編集に対応し、Claude Tag からも生成できるようになった。OpenAI 側は Codex を ChatGPT デスクトップ（macOS / Windows）に統合し、GitHub PR レビューやインライン編集を取り込む。Google の Logan Kilpatrick は「ACE（Agentic Coding Environment）は IDE の正統な後継だ」と述べ、IDE から「エージェント常駐環境」へという各社共通の方向性を言語化した。Notion も顧客フィードバックから PR マージまでを回す agent-native な「Ship OS」を発表している。

  • ClaudeDevs (X) — Claude Code on desktop now has an in-app browser
  • ClaudeDevs (X) — Artifacts now support public sharing and multiplayer editing
  • Logan Kilpatrick (X) — The Agentic Coding Environment (ACE) is the successor to the IDE
  • Notion (X) — Introducing Ship OS: the agent-native way to ship software

リサーチ・ベンチマーク

評価の土台が揺らぐ — ベンチマーク監査と「モデルの価値観」研究

今週は「何をどう測るか」への問い直しが続いた。OpenAI の SWE-Bench Pro 監査（上記）は、業界が競って参照してきた指標の 3 割が壊れていたと突きつけ、5 か月で 2 度目のベンチマーク推奨撤回となった。一方 Anthropic は、Claude が表出する 3,000 超の価値観がモデル間・言語間でどう変わるかを 30 万件超の匿名会話から分析する研究を公開。honesty や warmth のような価値の現れ方がモデルや言語で揺れることを示し、能力ベンチとは別軸の「振る舞いの計測」に光を当てた。数字の性能競争と並んで、その数字の信頼性と、モデルが内面に持つ傾向の可視化が、研究の焦点として浮上している。

  • AlphaSignal — OpenAI retracts SWE-Bench Pro after finding 30% of tasks broken
  • Anthropic (X) — How the values Claude expresses vary across models and languages

業界動向・規制

統治・人事・普及 — 標準化機関の提言、Bernanke 起用、教育への無償開放

モデル競争の外側でも節目が並んだ。Hassabis の標準化機関提言（上記）に加え、Anthropic は Long-Term Benefit Trust の新メンバーに元 FRB 議長 Ben Bernanke を迎え、統治体制に金融・政策の重鎮を据えた。普及面では、Anthropic が全米 K-12 の認定教員へ有料級 Claude を無償開放する Claude for Teachers を発表し、各州の学習指導基準に対応した教材ライブラリを用意。カナダの AI 研究機関へ 1,000 万カナダドルの拠出も表明した。労働面では Altman が「これまでのところ AI は差し引きで雇用を生んでいる——予想より楽観的だった自分の見立てよりさらに雇用寄りだ」と述べ、能力向上の局面での雇用インパクトを巡る議論も続いている。

  • Anthropic (X) — Dr. Ben Bernanke appointed to the Long-Term Benefit Trust
  • Claude (X) — Introducing Claude for Teachers: free access for verified K-12 educators
  • Sam Altman (X) — so far at least, AI has been net job-creating

来週の注目ポイント

最大の観測点は GPT-5.6 の需給。Altman が予告した供給の「きしみ」が実際に出るか、Sol の伸びと ChatGPT Work の定着がどこまで続くかを見たい。価格戦争では、7 月 19 日で切れる Anthropic の Fable 5 延長・レート 50% 増がさらに継続されるかが焦点で、GPT-5.6 の価格攻勢への次の一手として動く可能性が高い。ベンチマークでは、SWE-Bench Pro 撤回を受けた「壊れていない後継指標」の提案や、Sol Ultra の数学証明に対する数学者コミュニティの検証・追試が続報になる。統治面では、Hassabis の標準化機関提言への各ラボ・規制当局の反応、Grok 4.5 無料枠（launch window は 7 月 15 日前後まで）後の利用動向も引き続き追う。
