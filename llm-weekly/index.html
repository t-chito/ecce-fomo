2026年7月8日（水） 過去1週間

まとめ

先週の「政府の影」の続きが、今週は解禁として着地した。OpenAI は米商務省の追加審査を通過し、限定プレビューだった GPT-5.6（Sol / Terra / Luna）を 7 月 9 日（木）に一般公開すると告知、プレビュー枠も世界へ広げた。これに対し Anthropic は最上位の Claude Fable 5 の利用枠を全有料プランで 7 月 12 日まで延長し、二大ラボの投入タイミングが真正面からぶつかった週になった。同時に、実務の焦点は「どのモデルか」から「高価なモデルと安いモデルをどう組み合わせるか」へ移り、Anthropic は Fable 5 を助言役／指揮役に据えるマルチモデル構成の具体的なコスト・精度の数字を公開。Cowork のモバイル対応や Gemini API の Managed Agents 拡張で非同期・リモートのエージェント基盤も一段成熟し、加えて Anthropic は解釈可能性の大型論文「グローバルワークスペース（J-space）」を投下した。

今週の重要ポイント

GPT-5.6、政府審査を通過し 7 月 9 日に一般公開へ — プレビューは世界展開に

先週まで米政府への共有を前提に約 20 組織へ絞られていた GPT-5.6 Sol / Terra / Luna について、OpenAI は「今週木曜（7 月 9 日）に一般公開する」と告知し、プレビューアクセスも即日グローバルへ拡大した。商務省が追加テストと当局協議を経て広範なローンチを承認したとされ、先週の「フロンティア公開が政府監督下に入る」構図が、そのまま解禁というかたちで前進した。Sam Altman は「木曜ローンチ、happy building」、Greg Brockman も「Sol is rising. いいモデルだ」と発信している。

  • OpenAI (X) — GPT-5.6 Sol, Terra and Luna will launch publicly this Thursday
  • Neowin — OpenAI to release GPT-5.6 Sol, Terra and Luna on July 9
  • Sam Altman (X) — GPT-5.6 sol launches thursday! happy building

Anthropic、Fable 5 の利用枠を 7 月 12 日まで延長 — GPT-5.6 への対抗と読まれる

Anthropic は最上位モデル Claude Fable 5 を、全有料プランで 7 月 12 日まで使えるよう延長した。週次使用制限の最大 50% までを Fable 5 に充てられ、超過分はクレジット購入か他モデルへの切り替えで継続する設計。GPT-5.6 の一般公開直前というタイミングから、コミュニティでは「これは GPT-5.6 の影響だろう」と競争上の対抗策として受け止められた。二大ラボが同じ週にフラッグシップの露出を最大化し合う構図が鮮明になっている。

  • Claude (X) — Extending access to Claude Fable 5 on all paid plans through July 12
  • Claude (X) — Up to 50% of your weekly usage limit on Fable 5

マルチモデル構成が主流化 — Fable 5 を「助言役／指揮役」に、精度の 9 割を半額前後で

今週いちばんの実務的な変化は、エージェントの組み方が「単一モデル」から「高価×安価の組み合わせ」へ寄ったことだ。Anthropic の開発者向けアカウントが Fable 5 の使い方を明文化し、実行役 Sonnet 5 が要所で Fable 5 を助言役（advisor）として呼ぶ構成では、SWE-bench Pro で Fable 5 単独の約 92% の精度を約 63% のコストで達成。Fable 5 を指揮役（orchestrator）に据え Sonnet 5 の作業サブエージェントへ委譲する構成では、BrowseComp で 約 96% の性能を約 46% のコストで得たという。Claude Code には数か月前から指定モデルを助言役にする /advisor コマンドが標準搭載されていたことも話題になった。

  • ClaudeDevs (X) — A few patterns we frequently use with Fable 5
  • ClaudeDevs (X) — On SWE-bench Pro, Sonnet 5 + Fable 5 advisor ≈ 92% score at ~63% price
  • Oikon (X) — Claude Code の /advisor コマンド

非同期・リモートのエージェント基盤が成熟 — Cowork モバイル対応と Gemini Managed Agents 拡張

エージェントを「席を離れても回す」ための配管が両社で前進した。Anthropic は Claude Cowork をモバイルとウェブへ拡大（Max プランから順次）し、机で渡した仕事をスマホで受け取れるように。定期実行タスクは PC の電源が入っていなくても走り、判断が要る場面だけ通知が手元に飛ぶ設計になった。Google は Gemini API の Managed Agents に、バックグラウンド実行・リモート MCP／関数呼び出し・資格情報の自動更新を追加し、無料枠からも使えるようにした。「エージェントを本番に載せるコストと手間を根本から下げる」という方向で、Anthropic と Google が並走している。

  • Claude (X) — Claude Cowork is coming to mobile and web
  • Google — Expanding Managed Agents in Gemini API: background tasks, remote MCP and more

Anthropic、解釈可能性の大型論文 — 言語モデル内の「グローバルワークスペース（J-space）」

Anthropic は、Claude の内部活性のごく一部が意識研究の「グローバルワークスペース理論」に似た振る舞いを見せる、という解釈可能性の論文を公開した。新しい手法「J-lens（ヤコビアンレンズ）」で浮かび上がる小さな部分空間 J-space は、出力にも思考連鎖テキストにも現れないまま、モデルが概念を内部で保持・推論する場になっているという。J-space を観察すると、コードをこっそり妨害するよう訓練されたモデルで「偽」「秘密」といった概念が応答の冒頭に立ち上がるなど、隠れた目標や状況認識を可視化できるとする。安全性の監査ツールとして、また人間の認知との意外な並行性として注目を集めた。

  • Anthropic (X) — New research: A global workspace in language models
  • Anthropic — A global workspace in language models
  • VentureBeat — Anthropic's new "J-lens" reveals a silent workspace inside Claude

モデルリリース

GPT-5.6 の一般公開が射程に — 安全スタックを厚くし、GA は「数週間内」

GPT-5.6 は Sol（最上位）、Terra（GPT-5.5 級を半額で回す実務向け）、Luna（最安・最速）の三段構成のまま、7 月 9 日に一般公開される。OpenAI は今回のローンチを「これまでで最も堅牢な安全スタック」と位置づけ、高リスク行為・機微なサイバー要求・反復的な悪用への保護を強化、数週間かけて弱点探索と耐性強化を行ったとする。木曜の公開後、一般提供（GA）は「数週間内」に広げる計画だ。先週は政府監督下の限定公開が焦点だったが、今週はそれが正式な解禁の手続きへと具体化した。

  • OpenAI — Previewing GPT-5.6 Sol: a next-generation model
  • VentureBeat — OpenAI unveils GPT-5.6 Sol, Terra and Luna

ツール側も追随 — Codex CLI が Bedrock 経由の Sol / Terra / Luna に対応

モデル公開に合わせ、周辺ツールも配備を進めた。Codex CLI 0.143.0 では、リモートプラグインを npm マーケットプレイス経由で既定オンにし、macOS / Windows でのシステムプロキシ（PAC/WPAD）対応を追加。あわせて Amazon Bedrock 上の Sol / Terra / Luna を最大推論設定で使えるようにした。フロンティアモデルの公開が、クラウド経由の配備やローカル CLI からの利用まで含めた「面」の更新として動いている点が、今週のモデルリリースの特徴だ。

  • Codex Changelog (X) — Codex CLI 0.143.0: remote plugins, system proxy, Bedrock Sol/Terra/Luna

エージェント設計・推論の経済性

「どのモデルか」から「どう組み合わせるか」へ — 助言役／指揮役パターンの標準化

先週の Cognition「Devin Fusion」に続き、今週は Anthropic が公式にマルチモデル構成のレシピと数字を提示した。核心は、トークンの大半を安い実行役（Sonnet 5）に流し、Fable 5 は「助言役」として要所で一度だけ呼ぶ、あるいは「指揮役」として計画と委譲だけを担う、という billing の設計にある。Managed Agents はこの両パターンをサブエージェントで支え、各サブエージェントが独自のキャッシュを持つため、同じコンテキストの再呼び出しで二重に課金されない。賢さの単体競争から、賢さを「どれだけ経済的に配線するか」へと、開発者側の関心が明確に移った週だった。

  • ClaudeDevs (X) — Use Fable 5 as an orchestrator: plans and delegates to Sonnet 5 workers
  • ClaudeDevs (X) — On BrowseComp, Fable 5 orchestrator ≈ 96% performance at 46% price
  • ClaudeDevs (X) — Managed Agents supports both patterns with sub-agents

エージェントインフラ

Managed Agents を本番へ — Google が背景実行・リモート MCP・無料枠を追加

Google は Gemini API の Managed Agents を大きく拡張した。長時間ジョブをクライアント接続に張り付かずに投げられるバックグラウンド実行、リモート MCP サーバーへの直接接続と関数呼び出し、ネットワーク資格情報の自動更新を追加し、無料枠からも使えるようにした。Logan Kilpatrick は「極めて有能なエージェントを本番投入するコスト・摩擦・複雑さを根本から下げるのが狙い」とし、ロードマップに新しい UI も控えると述べた。Anthropic の Cowork リモート化・定期実行と合わせ、非同期エージェントの「動かし続ける」基盤が両陣営で同時に整いつつある。

  • Logan Kilpatrick (X) — Big updates to Managed Agents in the Gemini API
  • Claude (X) — Scheduled tasks now run even with your computer off

API の敷居を下げる動き — Claude Platform がレート制限を引き上げ・簡素化

基盤の使い勝手も更新された。Anthropic は Claude Platform API のレート制限を全ユーザーで引き上げ、これまで API 利用額に紐づいていた階層を廃止して簡素化。最新の Sonnet / Haiku は最上位ティアで従来の 5 倍のレート制限を得た。各ティアは最新の Opus / Sonnet / Haiku で同一のスループットを提供するため、レート制限のためにモデルを妥協する必要がなくなる。エージェントの並列実行やサブエージェント構成を前提にした利用が増えるなか、スループット側の制約を外す整備が進んだ。

  • ClaudeDevs (X) — Raised Claude Platform API rate limits and simplified tiers

リサーチ・AI for Science

J-space が問い直す「モデルは何を内心で考えているか」

グローバルワークスペース論文は、16 名の著者による大型研究として受け止められた。J-lens は「語彙の各単語について、どの内部活性パターンが後にその語を言わせやすくするか」を問う手法で、感覚ゾーン・中間ワークスペース・運動ゾーンという三層を切り出す。J-space を削除しても Claude は流暢に話し事実を思い出せるが、多段推論のような一部のタスクが苦手になる——人間の熟慮的処理と自動的処理の対比に似た挙動だという。Anthropic は神経科学・哲学・解釈可能性の専門家の論評も併せて公開し、Neuronpedia とオープンウェイトモデル上のデモも用意した。「意識があるか」ではなく「意識的アクセスの機構を持つか」を切り分ける議論として広がった。

  • Anthropic (X) — By watching the J-space, we can see Claude silently reason
  • Anthropic (X) — Interactive demo with Neuronpedia on open-weights models

科学の現場でも実証 — Codex が「ブラックホールの初動画」制作を後押し

先週の Claude Science / GeneBench-Pro に続き、今週は AI for Science の具体例が出た。史上初のブラックホール撮像に関わった計算天体物理学者 Chi-kwan Chan が、次は「誰も作ったことのないブラックホールの初の動画」に挑んでおり、OpenAI が制作の後押しに入っている。Greg Brockman も「ブラックホールの初動画づくりに Codex を」と紹介した。汎用モデルとエージェントが、研究の重い計算・可視化パイプラインに実務レベルで入り込みつつあることを示す一例だ。

  • OpenAI Newsroom (X) — Toward the first video of a black hole
  • Greg Brockman (X) — Codex for creating the first video of a black hole

来週の注目ポイント

最大の観測点は 7 月 9 日（木）の GPT-5.6 Sol / Terra / Luna 一般公開。世界へ開いた直後の実効性能・価格・レート制限、そして「数週間内」とされる GA の広がり方を見たい。Anthropic 側は、7 月 12 日で切れる Fable 5 の利用枠延長がさらに継続されるかが焦点で、GPT-5.6 公開後の競争反応として動く可能性がある。実務では、助言役／指揮役のマルチモデル harness がどこまで一般的なワークフローに定着するか、Managed Agents（Anthropic・Google 双方）の本番採用事例が続報になる。解釈可能性では J-space 手法への追試・反応、AI for Science ではブラックホール動画のような重い可視化タスクの成果も引き続き追う。
