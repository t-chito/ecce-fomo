LLM ニュース

2026年6月24日（水） 過去1週間

まとめ

今週の LLM 業界は「エージェントが製品の中心レイヤーへ移る」流れが一段と鮮明になった。Anthropic は Claude Code を発展させた Claude Tag を投入し、Slack のチャンネルに常駐して自律的にタスクをこなす「チームメイト」型のエージェントを打ち出した。同社は社内プロダクトチームのコードの 65% をこの内製版が書いていると公表している。OpenAI は防衛セキュリティに振り切った Daybreak を拡張し、GPT-5.5-Cyber 正式版・Codex Security・Patch the Planet で「機械の速度で脆弱性を塞ぐ」構想を具体化した。人材面では Google DeepMind から二大流出が起き、Transformer 共著者の Noam Shazeer が OpenAI へ、AlphaFold で 2024 年ノーベル化学賞の John Jumper が Anthropic へ移籍する。Grok は /goal で長時間自律実行、Gemini は Spark で常駐エージェントと、各社が同じ方向に動いた一週間だった。

今週の重要ポイント

Anthropic、Claude Tag を発表 — Slack に常駐する「チームメイト」型エージェント

Anthropic が Claude Code を発展させた Claude Tag を公開。Slack のチャンネルにメンバーとして加わり、タグ付けでタスクを委譲すると段階分解して自律的に処理し、PR の作成・マージやデータ分析、インシデント対応までスレッド内で完結させる。個人ではなくチーム共有のエージェントで、独自のアイデンティティと記憶を持つのが特徴。Opus 4.8 上で動き、Team / Enterprise プランのベータで提供される。

  • Claude (X) — Introducing Claude Tag, a new way for teams to work with Claude
  • TechCrunch — Anthropic's Claude Tag is learning your company, one Slack message at a time

OpenAI、Daybreak を拡張 — GPT-5.5-Cyber 正式版とオープンソース防衛

OpenAI がセキュリティ特化プログラム Daybreak を拡張。信頼できる防御者向けに GPT-5.5-Cyber 正式版を投入し、CyberGym で 85.6%（GPT-5.5 は 81.8%）の新 SOTA を主張。Codex 内で脆弱性を発見・検証・修正する Codex Security プラグイン、cURL・Python・Go など主要 OSS の保守者を支援する Patch the Planet、Cloudflare・CrowdStrike など 20 社超が参加する Cyber Partner Program を同時に打ち出した。

  • OpenAI — Daybreak: Tools for securing every organization in the world
  • SiliconANGLE — OpenAI expands Daybreak with Patch the Planet and full GPT-5.5-Cyber release

Google DeepMind から二大流出 — Shazeer が OpenAI、Jumper が Anthropic へ

同じ週に DeepMind の中核人材が二人、それぞれライバルへ移籍することが明らかになった。Gemini 共同リードで「Attention Is All You Need」共著者の Noam Shazeer が OpenAI へ。Sam Altman は「OpenAI 創業当初から最も一緒に働きたかった一人。10 年かかった」と歓迎した。さらに AlphaFold で 2024 年ノーベル化学賞を受けた John Jumper が、約 9 年在籍した DeepMind を離れ Anthropic へ移ることを表明。Demis Hassabis は「AlphaFold で成し遂げたことは世界を変えた」と謝意を述べた。

  • Benzinga — Google Gemini Co-Lead Noam Shazeer Joins OpenAI
  • CNBC — John Jumper to leave Google DeepMind for Anthropic

各社が「自律実行」を前面に — Grok /goal、Gemini Spark

常駐・自律で動くエージェントへ各社が一斉に寄せた。xAI（SpaceXAI）は Grok Build に /goal を追加し、複数ラウンドのサブエージェントが単一ゴールを実装・検証しながら長時間タスクを自律実行する仕組みを公開。Google は「あなたの 24/7 のパーソナル AI エージェント」と称する Gemini Spark を訴求し、DevRel の Logan Kilpatrick は「スーパーアプリ時代に入った感覚」と評した。Claude Tag と合わせ、エージェントの形が「対話ボット」から「常駐する実行主体」へ移る週となった。

  • Grok (X) — Introducing /goal in Grok Build
  • Logan Kilpatrick (X) — feels like we are entering the super app era

エージェントの新形態

Claude Tag — 「ボット」ではなく多人数で共有する実行主体

Anthropic の Boris Cherny は、Claude Tag を「単なる Slack のボットではない」と説明する。プロアクティブ（自分から動く）・マルチプレイヤー（チーム共有）・独自のアイデンティティと記憶を持つ点が従来のチャット型と異なり、チャンネルを追ううちに業務文脈を蓄積するため、毎回ゼロから説明する必要がない。アンビエント動作をオンにすると、止まったスレッドのフォローや他チャンネルからの関連情報の提示など、依頼を待たずに介入する。Anthropic 自身、社内プロダクトチームのコードの 65% をこの内製版が書いており、Claude Tag を構築したコードの大半もそれが書いたと公表している。

  • Boris Cherny (X) — We're launching Claude Tag today; proactive, multiplayer, with its own identity and memory
  • Claude Developers (X) — Claude Tag now writes 65% of our product team's code
  • Fortune — Anthropic launches Claude Tag, a tool that works like a virtual employee within Slack

Grok と Gemini も常駐・長時間自律へ

xAI は /goal に加え、Word プラグインや HeyGen・Interactive Brokers などのコネクタを相次いで投入し、Grok を業務やデータの上に常駐させる方向を強めた。Google は Gemini Spark を「ユーザーの指示の下で最初から最後まで重い作業をこなす常駐エージェント」と位置づけ、Gemini 3.5 Flash では Gmail から旅程を読み取って時差ボケ対策の睡眠スケジュールをカレンダーに登録するデモを示した。エージェントを「呼び出して使う道具」から「常に控える担当者」へと寄せる設計思想が、プロバイダ横断で共通している。

  • Gemini (X) — Gemini Spark is your 24/7 personal AI agent
  • xAI (X) — Grok models are now available on Databricks Agent Bricks

AI ×セキュリティ

「機械の速度で脆弱性を塞ぐ」 — Daybreak が発見から修正・パートナー網まで

Daybreak の今回の拡張は、脆弱性の「発見」だけでなく「修正の実装とマージ」までを射程に入れた点が新しい。OpenAI のモデルは主要ブラウザ・ネットワーク機器・OS の重大脆弱性に対してパッチを生成し始めており、Patch the Planet は Trail of Bits・HackerOne らと組んで研究者に Codex Security とモデルを供給し、保守者と直接協働する。30 超のプロジェクトが参加表明し、初期には cURL・Go・Python・Sigstore などが名を連ねる。Sam Altman は米政府（USG）やセキュリティ業界と連携して「すべての企業を守る」と述べ、防衛特化モデルを国家・産業のインフラ防護に結びつける姿勢を見せた。

  • OpenAI — Patch the Planet: a Daybreak initiative to support open source maintainers
  • Sam Altman (X) — We want to help all companies be secure, working with the USG and the security ecosystem

コーディングエージェントにセキュリティ審査が標準装備へ

セキュリティを開発フローに常設する動きは OpenAI 以外にも広がる。Cognition は Devin Review にセキュリティレビューを標準で組み込み、PR を開くたびにスキャナが見逃す脆弱性を自動検出し、各々の説明と修正案を起草する構成にした。コード生成エージェントが「速く書く」段階から、「書いたものが安全か検証する」段階を内蔵する方向へ進んでおり、GPT-5.5-Cyber のような防衛特化モデルがその裏側を支える構図が見えてきた。

  • Cognition (X) — Security review is now part of every Devin Review
  • OpenAI (X) — Expanding Daybreak: Codex Security plugin and full GPT-5.5-Cyber

AI for Science・ロボティクス

OpenAI、科学・医療での実証を相次いで公開

OpenAI は今週、科学領域での実証を畳みかけた。Boston Children's Hospital と Harvard との共同研究を NEJM AI に発表し、o3 Deep Research が未解決の希少小児疾患症例の再検討を助けたと報告。生命科学研究支援を測る新ベンチマーク LifeSciBench（173 名の科学者が関与、750 問）を公開し、GPT-5.4 が文献調査から検証済みの実験結果まで創薬プロジェクトを駆動した事例も示した。さらに健康相談で GPT-5.5 Instant が上位の Thinking モデルと同等に達したとし、週 2.3 億人が ChatGPT に健康・ウェルネスを尋ねる現状を踏まえ、緊急性の認識を強化したと説明した。

  • OpenAI (X) — NEJM AI study: o3 Deep Research helps revisit unsolved rare pediatric cases
  • OpenAI (X) — Introducing LifeSciBench

Anthropic Project Fetch Phase 2 — Opus 4.7 が人間の約 20 倍速でロボ犬を制御

Anthropic の Frontier Red Team が Project Fetch の第 2 フェーズを公開。ロボ犬（robodog）を Claude にプログラムさせる課題で、Opus 4.7 単独が前年の最良人間チーム（Opus 4.1 補助）より約 20 倍速く、人間が完了した全課題を 10 倍以上速くこなしたうえ、約 10 分の 1 のコード量で同等以上の成功率を出した。ただしビーチボールを「取ってくる（fetch）」精密動作には依然苦戦しており、自律 AI が物理世界でどこで破綻するかを測る安全研究としての性格が強い。

  • Anthropic — Project Fetch: Phase two
  • Anthropic (X) — Phase 2 of Project Fetch: Opus 4.7 ~20x faster than last year's best human team

開発ツール・インフラ

Claude Code に Artifacts と双方向 Design 連携

Anthropic は Claude Code に Artifacts を追加。セッションから PR ウォークスルーや常時更新されるダッシュボードなどのインタラクティブなページを生成し、社内限定リンクでチーム共有できる（Team / Enterprise ベータ）。あわせて Claude Design と Claude Code が双方向に同期するようになり、/design-sync でデザインシステムをリポジトリに取り込んで実コンポーネントに対して開発したり、作ったものを Design 側へ戻してキャンバスで編集し続けられる。MCP では Enterprise-Managed Auth 拡張に対応し、管理者が組織のコネクタを一括認可できるようにした。

  • Claude (X) — New in Claude Code: Artifacts
  • Claude Developers (X) — Claude Code and Claude Design now sync both ways

Codex は「実演で教える」へ、xAI は SpaceXAI として企業データ基盤に接続

OpenAI の Greg Brockman は、Codex を実演（demonstration）で教えられるようになったと示し、アプリの全機能テストや iOS アプリ開発の改善例を相次いで投稿。「ソフトウェアエンジニアリングは半年前すら思い出せないほど変わった」と述べた。リアルタイム音声の GPT-Realtime-2 も予告された。xAI 側は SpaceX との統合後「SpaceXAI」のブランドで、Grok モデルを Databricks Agent Bricks から企業データに接続し、エンタープライズのエージェント基盤に食い込む動きを見せている。開発・実行の主役がエージェントへ移る前提で、各社が配備面（IDE・企業データ・音声）を押さえに来た一週間だった。

  • Greg Brockman (X) — you can now teach Codex by demonstration
  • Greg Brockman (X) — software engineering is so different now
